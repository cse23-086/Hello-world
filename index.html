World Wide Web (WWW): The WWW is a global system of interconnected documents and resources on the internet. Users can browse and access web pages that contain text, images, and multimedia.

HTML (Hypertext Markup Language): HTML is the standard language for creating the structure and content of web pages. It uses tags to define elements like headings, paragraphs, links, and multimedia.

CSS (Cascading Style Sheets): CSS is a style sheet language that controls the presentation and layout of HTML documents. It separates content from design, allowing developers to apply consistent styles to elements like colors, fonts, and spacing.
